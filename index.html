Getting Started

Welcome to FlexiChart.js documentation! To get started, you first need to include FlexiChart.js in your HTML file...

<script src="https://cdn.jsdelivr.net/gh/praveensaneforceofficial/FlexiChart/FlexiChart.js"></script>

Bar Chart

This is an example of a bar chart:

var barChart = new FlexiChart("#chartContainer", { type: 'bar', data: [...], options: {...} });

Line Chart

This is an example of a line chart:

var lineChart = new FlexiChart("#chartContainer", { type: 'line', data: [...], options: {...} });
